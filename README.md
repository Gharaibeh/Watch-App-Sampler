A comprehensive tutorial app for WatchOS, in swift.

![](https://github.com/wangyanchang21/Watch-App-Sampler/blob/master/image/watch.jpg)


## Prerequisites

- Xcode 10+
- iOS 10 or later
- watchOS 4.0 or later
